
El Proyecto consite en un parent(challenge-libros) y 2 modulos.
Esta creado con Spring boot, y maven.

    -libros-mcsc: modulo de microservicios, se incluyen las librerias de spring web, JPA y
        H2 como BD para test.
    -libros-front: modulo de frontend, creado con react y thymeleaf.

para ejecutar el proyecto, se debe deployar en un tomcat o ejecutar directamente desde IDE.
